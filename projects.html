Password Manager is a commandline interface that locally stores passwords. The passwords are encrypted using AES256. Passwords can also be randomly generated using alpha-numeric and symbol combinations. Included is integrated credentials entry to quickly and accurately enter username s and passwords.

Integral Data Compiler

Calculates the estimated integral aeas under curves using Riemann sums of numerous equations ranging from linear functions to trigonometric equations. Creates a spreadsheet displaying x, y, and area values. Used to compare inaccuacies between left, right, and midpoint areas.

2D Linear Defense Game

Uses Java's swing library to create a GUI with the graphics library to render 2-Dimentional ge geometric graphics and hitbox detection. Enemies are randomly spawned offscreen with different velocities. Has a player AI which uses the greedy algorithm to find the highest priority target at a given time.
